Industry 4.0

Smart Industry of Industry 4.0 maakt wat men noemt een "smart factory". Binnen deze slimme fabrieken, worden machines gekoppeld en uitgelezen, zodat er een goed inzicht kan worden verkregen in wat er werkelijk in de fabriek gebeurt en de processen sneller kunnen worden geoptimaliseerd.

Web socket

Doormiddel van een Websocket is het mogelijk om live data van de client en de server te communiceren. Deze communicatie met de server word ook automatisch doorgevoerd naar de andere clients.

Het project

Het project draait om industrie 4.0. Het is namelijk een fabricagelijn die aangesloten kan worden op een selectie van Festo MPS modules. Op de computer kan via de webinterface geselecteerd worden welke extra handelingen je wilt uitvoeren op de potjes. De eerste optie is of je er een muntje in het potje wilt (via de MPS Handeling). De tweede optie is of je er een deksel op wilt (via de MPS Handeling) en als laatste kan je zeggen in welk magazijn je het potje wil laten eindigen (via de MPS Sorting).

Dit wordt allemaal aangestuurd via een Raspberry Pi 4 (master) en een siemens PLC. Het magazijn kan 3 verschillende soorten potjes aangeven (Rode, Zwarte, Zilveren). Op een computer kan je aangeven welke kleur je wilt ontvangen en de hoeveelheid.

Op de Settings pagina is het ook mogelijk om Dev-modus aan te zetten, hierdoor is de onderhouds pagina beschiktbaar.
